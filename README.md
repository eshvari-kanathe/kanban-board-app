
# Kanban Board App

A simple Kanban-style task management application that allows users to create, organize, and track tasks.

## Getting Started
Follow these steps to get a local copy running.

### Prerequisites
- Node.js (v18 or later)
- React (v19 or later)
- npm

### Installation

Clone the repository:
```bash
git clone https://github.com/eshvari-kanathe/kanban-board-app.git
```

Navigate to the project directory:
```bash
cd kanban-board-app
```

Install NPM packages:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```
